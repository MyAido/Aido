# Marketplace Quick Start Guide

## 🚀 Getting Started with Aido Marketplace

The Marketplace is your gateway to thousands of community-created AI commands. Here's how to use it!

---

## 📱 Accessing the Marketplace

### From Home Screen:
1. Open Aido app
2. Tap **"Marketplace"** in the top bar
   - OR -
3. Scroll down to **"Explore Marketplace"** quick action card
4. Tap **"Browse marketplace"** button

---

## 🔍 Browsing Preprompts

### Featured Commands
When you first open the Marketplace, you'll see **Featured Commands** - the most popular and high-quality preprompts curated by the community.

**Currently Featured:**
- `@translate` - Translate text between languages (15.4K downloads)
- `@code` - Generate code snippets (12.8K downloads)  
- `@meeting` - Summarize meeting notes (9.8K downloads)

### Search for Specific Commands
1. Tap the **search bar** at the top
2. Type keywords like "email", "coding", "creative"
3. Results update in real-time
4. Clear search with the ❌ button

### Filter & Sort
Tap the **filter icon** (funnel) to:
- **Browse by Category:**
  - Writing, Communication, Productivity
  - Technical, Creative, Business
  - Education, Social, Fun, General
  
- **Sort Options:**
  - Popular (most downloads)
  - Top Rated (highest ratings)
  - Newest (recently added)
  - Trending (hot right now)

- **Quality Filters:**
  - Minimum rating (0-5 stars)
  - Featured commands only
  - Verified creators only

---

## ⬇️ Installing Commands

### To Install a Preprompt:
1. Find a command you like
2. Read the description and example usage
3. Check ratings and download count
4. Tap **"Install Command"** button
5. ✅ Done! Use it immediately with the trigger

**Example:**
```
Install @translate from marketplace
Then type: "Bonjour le monde @translate to English"
Result: "Hello world"
```

### What Happens When You Install?
- Command is added to your local preprompts
- Available in all apps via Accessibility Service
- Appears in your Commands list
- No internet needed after installation (unless using Gemini API)

---

## ⭐ Managing Favorites

### Add to Favorites:
1. Tap the **heart icon** ❤️ on any preprompt card
2. Access later in **"Favorites"** tab
3. Quick-install favorite commands on new devices

### Remove from Favorites:
1. Go to **"Favorites"** tab
2. Tap the **filled heart icon** ❤️ again
3. Command removed from favorites (but stays installed locally)

---

## 📤 Sharing Your Commands

### Upload Your Custom Preprompt:
1. Go to **"My Shared"** tab
2. Tap **"Share New Command"** button
3. Fill in details:
   - **Category:** Choose the best fit
   - **Description:** Explain what it does
   - **Tags:** Add searchable keywords
4. Tap **"Share"**
5. 🎉 Your command is now public!

### Managing Your Shared Commands:
- View download counts
- See ratings and reviews
- Update descriptions
- Delete if needed

**Tips for Success:**
- ✅ Write clear descriptions
- ✅ Add relevant tags
- ✅ Include example usage
- ✅ Choose correct category
- ✅ Test before sharing

---

## 🔄 Cloud Sync

### What is Cloud Sync?
Automatically syncs your preprompts across all your devices.

### Enable Sync:
1. Sign in with your account (or continue as guest)
2. Tap the **sync icon** 🔄 in Marketplace
3. Your local commands upload to cloud
4. Download commands from other devices

### Sync Status:
- 🔄 **Syncing** - In progress
- ✅ **Synced** - Up to date
- ⚠️ **Error** - Check connection

### Automatic Sync:
Runs every hour in the background when:
- Connected to WiFi (recommended)
- Battery above 20%
- App is installed

## ⭐ Rating & Reviewing

### Rate a Preprompt:
1. Open preprompt details (tap card)
2. Tap star rating (1-5 stars)
3. Optionally add written review
4. Submit

**Rating Guide:**
- ⭐⭐⭐⭐⭐ (5) - Excellent, works perfectly
- ⭐⭐⭐⭐ (4) - Very good, minor issues
- ⭐⭐⭐ (3) - Average, needs improvement
- ⭐⭐ (2) - Poor, rarely useful
- ⭐ (1) - Broken or misleading

## 🎯 Understanding Preprompt Cards

### Card Information:
```
┌─────────────────────────────────┐
│ @trigger ✓                     ❤│ ← Trigger + Verified badge + Favorite
│ Description text...              │ ← What it does
│ [Category] #tag1 #tag2          │ ← Category chip + tags
│ 👇 15.4K    ⭐ 4.8 (342)        │ ← Downloads + Rating (count)

```

### Badges:
- **✓ Verified** - High-quality, tested command from trusted creator
- **🌟 Featured** - Highlighted by Aido team

## ❓ Troubleshooting

### "Command already exists"
**Solution:** You already have a preprompt with this trigger. Edit or delete existing one first.

### "Sync failed"
**Solutions:**
- Check internet connection
- Sign in to your account
- Try manual sync (tap refresh icon)

### "No results found"
**Solutions:**
- Try different search terms
- Remove filters (reset to "All")
- Browse by category instead

### Can't find installed command
**Solutions:**
- Check Commands screen (not Marketplace)
- Ensure Accessibility Service is enabled
- Restart Aido service in Settings
