## 🚀 What is Aido?

Aido is not just another keyboard or chatbot. It is a **smart accessibility overlay** that lives on top of your existing apps (WhatsApp, Slack, Gmail, etc.). 

By using Android's **Accessibility Services**, Aido can read the text you are typing or viewing and offer instant AI assistance without you ever leaving the app.

## ✨ Key Features

### ✍️ AI Writer
Generate emails, essays, code snippets, or creative stories instantly. just type a prompt and let Aido finish it.

### 💬 Smart Replies
Aido analyzes incoming messages and suggests context-aware replies. Reply professionally, casually, or humorously with a single tap.

### 🔤 Font Library (New!)
Transform your text into over **1000+ unique styles** (e.g., 𝐁𝐨𝐥𝐝, 𝑆𝑐𝑟𝑖𝑝𝑡, 𝙼𝚘𝚗𝚘).
- **Trigger-based:** Type `@b` to switch to Bold.
- **Custom Fonts:** Create your own mappings or import `.aidofont` files.
- **[Browse Font Library](https://myaido.web.app/font-library.html)**

### 🛠️ Utility Commands
Powerful shortcuts for everyday tasks:
- `.date` - Insert current date.
- `.calc` - Solve math expressions in-place.
- `.tr:es` - Translate text to Spanish instantly.

## 🔒 Privacy & Safety

Aido is built with a **Privacy-First** architecture:
*   **No Keylogging:** Aido does *not* store your keystrokes. It only processes text when you explicitly trigger an action.
*   **Local Processing:** Many features run locally. API calls are ephemeral and not stored.
*   **Transparency:** We are fully transparent about our use of Accessibility Services. [Read our Transparency Hub](https://myaido.web.app/transparency/).
